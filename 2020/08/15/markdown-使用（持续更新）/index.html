markdown 视频插入方法

利用Iframe标签插入网络视频

<iframe     
        height=450      
        width=800      
        src="视频地址"      
        frameborder=0      
        allowfullscreen> 
</iframe>

本地视频插入方法

<video src="./test.mp4" width="800px" height="600px" controls="controls"> </video>

图片不显示的的解决方法

引入图片失败的时候，F按12 提示 403错误

在每个.md 文件的头部加上这句话，就可以解决网络图片引入失败的问题

<meta name="referrer" content="no-referrer" />
