总结

npm虽然下载慢，但是node_modules文件夹的文件少(是被合并到其他文件里面了)，package.json的 dependencies(项目依赖)里面的文件目录比较清晰，而且会生成package-lock.json。

cnpm除了速度比npm快，其他都没有npm好。

这个事情也是在学校校园网用不了npm才发现的
